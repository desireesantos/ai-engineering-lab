# AI Engineering Lab

A collection of daily experiments combining:

- AI-assisted engineering
- Developer productivity
- Software architecture
- Engineering workflows
- Technical leadership

This repository is focused on exploring how AI can improve software engineering processes, developer experience, and engineering decision-making.

## Current Topics

- AI-assisted PR reviews
- Engineering productivity
- Risk analysis
- Architecture insights
- Developer workflows
- Automation experiments

## Stack

- TypeScript
- React
- Ruby
- Node.js
- AI APIs
- GitHub Actions

## Goal

Build small daily experiments to explore practical applications of AI in modern engineering teams.
